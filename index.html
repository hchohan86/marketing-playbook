01 Set up a 5-Email Lead Nurture Drip Campaign in Mautic
Campaign Mgmt Medium ▼
🛠 Mautic (free) · LibreOffice · Browser Universal — Any Business

Set up a 5-email lead nurture sequence for new leads from a product landing page. Verifiable output: active campaign with correct send intervals and zero errors.

  • Log in to Mautic → Contacts → Segments. Create segment "New Landing Page Leads" filtered by source tag.
  • Go to Emails → New. Build Email 1 (Welcome) using drag-and-drop builder. Set subject, body, CTA button.
  • Repeat for Emails 2–5 (Day 3, 7, 14, 21 content with escalating CTAs).
  • Go to Campaigns → New Campaign. Name it "Product Nurture Sequence". Add source segment.
  • Add Action: Send Email 1 on entry. Add Decision: Wait 3 days → Send Email 2. Repeat for each.
  • Add goal: contact visits pricing page (URL condition). Set campaign to Published.
  • Send test emails to internal address. Verify rendering on mobile and desktop.
  • Monitor Campaign → Statistics after first trigger fires. Confirm 0 errors, first email queued within 10 min.
03 Produce a Monthly Campaign Performance Report from GA4
Market Analysis Medium ▼
🛠 Google Analytics 4 · LibreOffice · Browser Universal — Any Business

Build a Month-on-Month analytics report covering traffic, conversions, and channel attribution. Delivers a 4–6 page PDF by month-end.

  • Log in to GA4 → Reports → Acquisition → Traffic Acquisition. Set current vs. prior month date range. Export CSV.
  • Open LibreOffice Calc. Paste CSV. Create columns: Channel, Sessions, Bounce Rate, Avg Duration, Conversions, Conv. Rate.
  • Add MoM % change formulas for each metric.
  • Export GA4 Conversions and Campaigns data. Add to additional spreadsheet tabs.
  • Create summary tab: top 3 and bottom 2 campaigns by conversion rate.
  • Write 5-bullet insight section: growth, drops, anomaly, recommended action.
  • Format and export entire report as PDF. Deliver same day as month-end.
06 Build a Monthly Marketing Budget Tracker with Variance Flags
Budget Mgmt Medium ▼
🛠 LibreOffice Calc (free) Universal — Any Business

Build a Planned vs. Actual spend tracker with overrun alerts, colour-coded status columns, and a bar chart summary. Zero formula errors required.

  • Open LibreOffice Calc. Sheet 1 "Budget Tracker": Category, Channel, Planned Budget, Actual Spend, Variance ($), Variance (%), Status.
  • Add rows: Paid Social, Paid Search, Email, SEO/Content, Events, Tools/Software, Agency Fees, Miscellaneous, TOTAL.
  • Variance ($) = Actual – Planned. Variance (%) = IF(Planned>0, (Actual–Planned)/Planned, 0).
  • Status column: IF Variance% >10% → "OVER", >0% → "AT RISK", else "ON TRACK".
  • Conditional formatting: red=OVER, amber=AT RISK, green=ON TRACK. TOTAL row uses SUM formulas.
  • Sheet 2 "Monthly Summary": pull totals, create bar chart (Planned vs. Actual by channel).
  • Protect formula cells. Save as .xlsx. Share with finance team.
09 Run an SEO Keyword Gap Analysis vs. a Competitor
Market Analysis Medium ▼
🛠 Screaming Frog (free ≤500 URLs) · Google Search Console Universal — Any Business

Identify ranking opportunities your site is missing while a competitor wins. Output: XLSX with 20 prioritised keywords and recommended page actions.

  • Open Screaming Frog. Enter your domain. Crawl. Export: All URLs, Page Titles, Meta Descriptions, H1, Word Count.
  • Open Google Search Console → Performance. Export all queries (last 3 months): Query, Clicks, Impressions, CTR, Position.
  • Open LibreOffice Calc. GSC queries in Sheet 1. Screaming Frog data in Sheet 2. Filter queries Position 11–30 (near-miss).
  • Search Google for site:competitor.com. Manually note 10–15 top-ranking URLs. Check if your site appears for the same keyword.
  • Create gap table Sheet 3: Keyword, Competitor URL, Competitor Position, Your Position. Score by relevance. Highlight top 20.
  • Export as XLSX. Write a 1-line recommended action per keyword (new page, update existing, internal link).

(Step-by-Step)
1. Define 5 direct competitors. For each: record homepage URL, pricing page, and 3 recent ad creatives from Meta Ad Library.

8. Export as PDF. Present to stakeholders with 3 recommended positioning adjustments.
